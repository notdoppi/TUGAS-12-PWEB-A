# TUGAS-12-PWEB-A
Pada tugas ini kita diminta untuk membuat web yang bisa untuk mengupload foto/gambar.

## Deskripsi
Website ini adalah sebuah sistem informasi yang dirancang untuk mengelola data siswa. Website ini menerapkan konsep CRUD (Create, Update, Read, Delete) yang memfasilitasi pengguna untuk melakukan perubahan data secara lengkap melalui website.

fitur utama dari Website ini adalah fitur untuk mengupload dan menyimpan file gambar siswa ke dalam direktori server dan untuk informasi datanya disimpan di dalam database.

## Tujuan
- Penyimpanan menjadi lebih terstruktur
- Efisiensi data
- verifikasi visual dengan memungkinkan penyimpanan foto yang terhubung dengan data identitasnya

## Fitur utama
- Tambah data (Create)
- Menampilkan data (Read)
- Ubah data (Update)
- Hapus data (Delete)

## Teknologi yang digunakan
- Bahasa pemrograman = PHP
- Database = MySQL
- Webserver = Apache (XAMPP)
- Frontend = HTML, CSS
- Tools Pendukung = VSCode, phpMyAdmin
